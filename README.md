# Movie & TV Show Popularity Analytics Dashboard

## 📌 Deskripsi Proyek

Proyek ini bertujuan untuk menganalisis faktor-faktor yang memengaruhi popularitas film dan TV show berdasarkan **genre, bahasa, tahun rilis, jumlah vote, dan rating pengguna**. Analisis dilakukan menggunakan Python untuk proses data cleaning, exploratory data analysis (EDA), visualisasi data, serta analisis statistik. Hasil analisis kemudian disajikan dalam dashboard interaktif menggunakan Tableau untuk memudahkan eksplorasi data dan penyampaian insight.

Proyek ini dikembangkan sebagai bagian dari assignment Data Analytics dengan menerapkan proses analisis data secara end-to-end, mulai dari preprocessing data hingga pembuatan dashboard interaktif.

---

## 🎯 Tujuan

- Mengidentifikasi genre dengan rata-rata popularitas tertinggi.
- Mengetahui genre yang memiliki jumlah film dan TV show terbanyak.
- Menganalisis tren popularitas berdasarkan tahun rilis.
- Mengidentifikasi distribusi bahasa pada konten populer.
- Membandingkan tingkat popularitas antara genre **Action** dan **Adventure** menggunakan analisis statistik.

---

## 📊 Analisis yang Dilakukan

### Exploratory Data Analysis (EDA)

- Rata-rata Popularity berdasarkan Genre
- Top 10 Genre berdasarkan Jumlah Film dan TV Show
- Tren Popularity berdasarkan Tahun Rilis
- Distribusi Original Language pada Konten Populer

### Statistik Deskriptif

- Mean Popularity
- Median Popularity
- Standard Deviation
- Nilai Minimum dan Maksimum Popularity
- Jumlah Film per Genre

### Statistik Inferensial

- **Independent Two Sample t-Test** untuk mengetahui apakah terdapat perbedaan rata-rata popularity antara genre **Action** dan **Adventure**.

## 📈 Dashboard
